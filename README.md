## Aviary project

1. Clone repository
2. Execute
```
npm install
```
3. Install Nest CLI
```
npm i -g @nestjs/cli
```
4. Rename file .env.template to .env
5. Start application in mode development
```
npm start run:dev
```
## Stacks
* postgreSQL
* Nest
* Docker
